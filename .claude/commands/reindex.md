# Reindex Search

Rebuild the search index from MDX documents.

Arguments: $ARGUMENTS

## Execution

1. Run the reindex command:
   ```
   npm run reindex
   ```

2. Report results:
   - Total documents indexed
   - Total chunks generated
   - Any errors or warnings

## Output

```
=== Reindex Complete ===

Documents: N
Chunks: M
Errors: K (if any)
```
