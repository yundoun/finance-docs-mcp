# 프로젝트 문서

기획 → 설계 → 운영 단계별 산출물을 관리한다.

```
docs/
├── planning/                    ← 기획 단계
│   ├── PROJECT-BLUEPRINT.md     ← Lean Canvas + IA + 로드맵 (완료)
│   ├── OPEN-QUESTIONS.md        ← 미결정 사항 + 결정 로그
│   ├── CONTENT-PLAN.md          ← 콘텐츠 우선순위 + 작성 계획 (Phase 1 확정 후)
│   └── COMPETITIVE-ANALYSIS.md  ← 경쟁 분석 (필요 시)
│
├── design/                      ← 설계 단계
│   ├── FRONTMATTER-SPEC.md      ← frontmatter 스키마 확정본
│   ├── CONTENT-TEMPLATE.md      ← MDX 문서 작성 템플릿
│   ├── QUALITY-CHECKLIST.md     ← 문서 품질 체크리스트
│   └── TAXONOMY.md              ← 카테고리/태그 분류 체계
│
└── operations/                  ← 운영 단계
    ├── CONTRIBUTING.md          ← 기여 가이드 (Phase 2)
    ├── REVIEW-PROCESS.md        ← 문서 리뷰 프로세스 (Phase 2)
    └── UPDATE-POLICY.md         ← 제도 변경 시 갱신 정책 (Phase 2)
```
